## Getting Started

### Prerequisites

- Node.js (^20.19.0 || >=22.12.0)

### Installation

1. Clone this repository
2. Install dependencies:
   ```bash
   npm install
   ```

### Development

Start the development server:

```bash
npm run dev
```

### Building

Build for production:

```bash
npm run build
```

Preview production build:

```bash
npm run preview
```

### Testing

Run tests:

```bash
npm test
```

Type checking:

```bash
npm run type-check
```
